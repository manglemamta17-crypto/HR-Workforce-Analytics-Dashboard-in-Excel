# HR Workforce Analytics Dashboard in Excel

This project is an interactive HR dashboard created in Microsoft Excel to analyze workforce data, staff distribution, gender ratio, and employee status across locations.

## Features

* Staff count by designation
* Location-wise employee distribution
* Gender analysis
* Active / Left / Leave status

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
